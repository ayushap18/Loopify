# Loopify

Loopify is a simple Chrome extension that lets you select, loop, and mash up your favorite parts of Spotify songs.

## How to Use

1. **Install**: Load the extension in Chrome (`chrome://extensions` → Developer mode → Load unpacked → select this folder).
2. **Connect Spotify**: Click the Loopify icon and connect your Spotify account (Premium required).
3. **Play a Song**: Open [Spotify Web Player](https://open.spotify.com) and play any track.
4. **Select a Loop**: Use the popup to auto-suggest or manually set start/end points, then click "Start Loop".
5. **Save Loops**: Save your favorite segments. Play them as a mashup or create a playlist.

## Requirements
- Spotify Premium account
- Chrome or Chromium-based browser

## Features
- Loop any part of any Spotify song
- Save multiple loops from different songs
- Play all saved loops as a mashup
- Create a playlist from your saved loops
